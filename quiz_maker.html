Quiz Maker

Created the front-end for a toy quiz webservice where registered users can make their own quizzes and host play sessions for others to join and take their quiz. The front-end is built using ReactJS.

User Dashboard

Create Your Own Quizzes: The user can set the question, answers, time limit and points for each question. They can also choose to embed an image or link to a YouTube video.

Question Form

Start a Session: The user can create play sessions for their quizzes. Players can join by using the session code.
